Update product variation on cart page.

== Description ==
Update product variation on cart page so customers don't need to remove and add product again.This plugin require WooCommerce. It will only work with standard woocommerce variable products.

== Installation ==

Download the plugin

e.g.

1. Upload plugin files to the '/wp-content/plugins/plugin-name' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Cart Variation Update screen to configure the plugin
